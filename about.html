Sobre o Projeto

Uma jornada científica e curiosa através do cosmos.

Propósito e Objetivos

Este projeto foi desenvolvido como um trabalho individual para a disciplina de Desenvolvimento Web, com o objetivo principal de aplicar os conhecimentos adquiridos em sala de aula na construção de um site dinâmico e interativo. A temática da NASA e do universo foi escolhida para explorar dados fascinantes e criar uma experiência envolvente para o usuário.

Os principais objetivos incluem:

  • Consumir diversas APIs públicas da NASA para exibir dados em tempo real.
  • Utilizar o framework Bootstrap para garantir um design responsivo e moderno.
  • Aplicar conceitos avançados de JavaScript como filter, findIndex, map, join, e o spread operator (...) para manipulação eficiente de arrays e dados.
  • Estruturar o site com múltiplas páginas, incluindo carrossel, galerias de imagens e seções de texto informativo.
  • Incorporar efeitos visuais como Parallax para uma experiência de usuário mais imersiva.

Tecnologias Utilizadas

Para a construção deste site, foram empregadas as seguintes tecnologias:

  • HTML5: Para a estrutura e conteúdo das páginas.
  • CSS3: Para estilização, incluindo customizações e o efeito Parallax.
  • JavaScript (ES6+): Para a lógica de consumo de APIs, manipulação do DOM e aplicação dos conceitos de array.
  • Bootstrap 5: Como framework CSS para layout responsivo, componentes de UI (navbar, carrossel, cards, etc.).
  • APIs da NASA:
    • Astronomy Picture of the Day (APOD)
    • Mars Rover Photos
    • NeoWs (Near Earth Object Web Service)
